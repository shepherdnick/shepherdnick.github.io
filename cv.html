Notable Achievements whilst in post:

Education

Ph.D. - Computer Science

2016, Swansea University

My PhD has concentrated on information retrieval of social media content, in particular microblog content. Working with an Industrial partner I have been researching what makes a tweet useful to a user performing some of the most common types of search tasks performed over microblogging corpuses (based off of research from Microsoft Research). Having run several experiments I was able to find features as to what made tweets useful, and not useful. Utilising crowdsourcing and machine learning to implement learning to rank, we were able to rank these features and produce a search engine which provides ‘useful’ search results to users.
