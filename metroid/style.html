YOU ARE IN: METROID PRIME: STYLE

This page will be me talking mainly about the creation of this web site.

I have a passion for web design.  I have not got much experience in the subject, and this is my biggest project to date (first time I have used frames).  Despite my friends opinion on HTML, I have chosen to code most of the web site in HTML.  As a pose to using "Flash" to create interactivity within the web-site I have opted to use HTML, as I think it can be just as interactive.

I based the theme of the web site on a game - there's the main story mode, a pause menu and an options menu, common in today's video games.  I wanted to have an original idea, and thought of something that no-one else would think of.  Even though the immediate interactivity like that of a game is not there, it is appropriate for the internet.

The navigation bar on the side <-- was my idea of trying to keep the viewer in contact with the rest of the web site.  As the viewer clicks on a new menu, like the Pause menu, it opens a page in the main view and changes the navigation bar.

Over the past month, I have spent over 48 hours in making the web site, which does seem like a long time, but I have to bear in mind, I have learned a lot of things, and had to change and update things so many times it makes my head hurt!  It is good to keep a log of what changes are actually made when making a web site.  I know with this one, the log is pages long!

I also spent a lot of the time on research.  Half way through making, the game was released - I couldn't base all of my content on me playing the actual game.  So I spent a long time scouring the internet for reviews and buying many Nintendo magazines for the pictures and reviews.  I hyped myself up for the release of the game, and was very delighted once I was able to play it over my friends house.

I have tried to keep the style interesting, for example, by changing the colour of the scroll bars (by a little JavaScript a friend gave to me).  It's also the first time I have used "Macromedia Flash" (and I have to say, I'm quite proud of the opening animation!).  I have learned so much whilst making the web site, and even if I don't win anything, it has still been a valuable lesson in web design for me.

I hope you have enjoyed reading the web site as much as I have in making it!
