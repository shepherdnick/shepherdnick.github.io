Web Design

From since a very long time ago, about 2 years ago I'd say I have always had an interest in the way that people manage and create web sites. One day I bought a book that taught me how to code in Hyper Text Markup Language or in HTML as other people may know it as. This Language enables you to make web pages. As soon as I had learned the basics, I was ready to unleash probably one of the only creative things I am able to do.

It started off when I themed my web site around this little guy:

His name is Tap Man and, well, you can read all about him if you go to the web site.

The other major web site that I made was for the school company that I was the Managing Director of. The company was called "Cool Fusion", and you can visit the site from the following link:

The last kind of basic HTML site that I made was one that I entered for a competition. I was making the web site about a game that was going to be released on the Nintendo GameCube, and it's name: Metroid Prime. The site works well, but only on a fast connection. This is because back then I didn't have the ability or knowledge to compress the image files.

After that, I was given the opportunity of having unlimited web space on my affiliates server (ToBeOn). I made a very poor web site, but graphically it was the most advanced I had been able to achieve. OK, click here to see the previously unreleased version of Shepherdnick.co.uk in it's very very early days!

Things progressed ever so slowly from there, until one day I decided to completely change the web site altogether. Instead of being graphically reliant, I changed it so that it was all coded beauty. And from 8th August 2003, Shepherdnick.co.uk has looked (roughly) the same.

I took inspiration for my site from many sources, many of which, Kat (my best HTML buddy) inspired me to make. I also used to visit web sites which taught me how to code in HTML in better and more complex ways until I was finally confident enough to try something for real.

From then on, I couldn't stop making web sites. Apart from my main site which I update as regularly as I can, I have made the following web sites! Hope you enjoy what you see!
