Name of site The Image Wall
URL http://www.imagewall.com/
Content The Image Wall is a collection of photographs and digital art, collaberated into a fellow web designers personal site.
Graphics Image has been spliced to make up the layout to the site. Even though the site is quite image intense, file compression has allowed for the images to load quickly, even on a 56k.
Layout Navigation is enhanced by the menus and links towards the left side of the screen. The random image adds a nice touch of interactivity and keeps the site fresh, by changing every time the user visits the site.
Colour Scheme Colours compliment each other. The quite bland grey/brown colour could be coupled with almost any other colour, but the red theme works very well. The bland grey/brown colour takes the nasty glow of the default white colour of browsers off of the screen.
Highlight Collection of Stock Images
Why I chose it It shows a good example of how web sites can be viewed without scrolling. Everything the user needs to see is already on the screen. I found it to be a very innovative web site with interesting content. Worth taking a look for 10 minutes.
