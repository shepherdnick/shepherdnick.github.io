This month's featured web site is:

Name of site Broots
URL http://home.no.net/broots/
Content Broots web site is mainly a high-tech/spec blog site, but contains links to affiliates and personal photos
Graphics Few images have been used throughout the site, but when they have they take on the same colour scheme as the rest of the site. Most of the graphics used are for links to relevant parts of the web site.
Layout All the different cells of the table are neatly lined up so that the whole layout of the web site fits onto one screen. IFrames make it so that the content is evenly distributed throughout the screen.
Colour Scheme The colour scheme was well thought out, and the shades of blue that were used are soft on the eyes. The colours also blend in well with each other, yet still defining the cells of the table.
Highlight World Wide Web News
Why I chose it It shows a good example of how web sites can be viewed without scrolling. Everything the user needs to see is already on the screen. I found it to be a very innovative web site with interesting content. Worth taking a look for 10 minutes.
